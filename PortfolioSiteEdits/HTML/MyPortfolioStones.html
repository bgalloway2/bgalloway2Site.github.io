Stones Candies Website and Online Ordering

This project is a website for a local candy store in Oswego, New York called Stones Candies. The idea of this website is to create a visually appealing website that allows users to search for different types of candy they would like, and purchase them online for home delivery. This project is still in the early design stages.

My Project Goal

This project is a collaboration with one other person in hopes to design a site that allows users to easily navigate through different candy types and purchase them. The main focus is on creating an e-commerce system that is secure for users to enter their card information and trust they will recieve what they order.

Home Screen

Candy Browser

This screen representers where users would filter different candy options and make the selections they would like. The idea is that when a user selects one of the candy images, it will enlarge with an option to increase the quanitity. The user can than add however many of the item to their cart for purchase.
